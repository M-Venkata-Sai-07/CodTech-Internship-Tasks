* Analyzed NYC Yellow Taxi dataset
* Performed data cleaning and exploration
* Extracted insights from large-scale data

**Key Insights:**

* Most trips involve 1–2 passengers
* Average trip distance is relatively short
* High fare trips are less frequent
* Certain payment methods dominate
